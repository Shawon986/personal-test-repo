Defination of Tag:

Tag is such a keyword or short form which defines how the web browser will format and display the content.👇

such as : <p>,<title>

There are 2 types of tag in html-👇
  1. Container tag
  2. Empty tag

Defination of Container Tag:

Container tag has something its inside and it has opening and closing.👇
Such as:<p>,</p>,<b>,</b>,<u>,</u>

Defination of Empty Tag:

Empty tag has only closing 👇
Such as: <br/>,<hr/>

Defination of Elements:

Html element is defined by a start tag ,content and an end tag. The element is everything from start to end tag.👇

Such as: <p> My cat is very grumpy<p/> (The whole thing is called element)

Defination of attribute:

Attribute is used for change the behaviour or display of an html element👇
Such as: Style is an attribute,it is used for styling any content
